## t3mujinpack Reference

**Source**: t3mujinpack — community B&W film emulation pack

Neopan 1600 is referenced with these defining characteristics:
- High-speed traditional B&W (discontinued)
- Grainy, contrasty, classic journalistic look
- Grain structure between Tri-X 400 and Delta 3200
- Grain: Amount 50, Size 45, Roughness 65

## Community Core Settings for Neopan 1600 Look

### Basic Panel

Neopan 1600 is high-contrast, grainy, and dramatic:

| Setting | Value | Notes |
|---------|-------|-------|
| Contrast | +30 to +40 | High native contrast |
| Highlights | -25 to -35 | Protect bright stage lights/skies |
| Shadows | +10 to +20 | Moderate shadow lift |
| Whites | +10 to +20 | Clean whites |
| Blacks | -25 to -30 | Deep, dramatic blacks |

### B&W Mix

Neopan 1600 has ortho-panchromatic sensitivity with red extension — classic journalistic rendering:

| Channel | Value | Notes |
|---------|-------|-------|
| Red | +15 to +25 | Brightens skin dramatically |
| Orange | +10 to +20 | Luminous skin under streetlights |
| Yellow | 0 to +10 | Warm highlight pop |
| Green | -10 to -15 | Darkens foliage for depth |
| Aqua | -5 to -10 | Cool tone definition |
| Blue | -20 to -30 | Dramatic sky darkening |
| Purple | -5 to -10 | Subtle background darkening |
| Magenta | 0 to -5 | Subtle magenta control |

### Grain (from STYLEGUIDE — between Tri-X and Delta 3200 territory)

| Setting | Value |
|---------|-------|
| Amount | 45-55 |
| Size | 40-50 |
| Roughness | 60-70 |

## Community Data Validation

### Validity Assessment: MODERATE

Neopan 1600 is a discontinued film, so community recipes are based on memory, scans, and comparison to still-available films. The t3mujinpack provides a useful reference point. The film's character is positioned between Tri-X 400 (ISO 400, pushed) and Delta 3200 — grainier than the former but less extreme than the latter.
