# Lomochrome Purple — Technical Datasheet

## Overview

Lomochrome Purple is a C-41 color negative film manufactured by Lomography, designed to produce surreal color shifts inspired by Kodak Aerochrome infrared film. It is **not** an infrared film — it achieves its look through modified dye couplers in the emulsion layers rather than IR sensitivity.

## Key Specifications

| Property | Value |
|----------|-------|
| **Film Type** | Color negative (C-41 process) |
| **ISO Range** | 100–400 (variable) |
| **Formats** | 35mm, 120, 110, 16mm (limited), Super 8 (limited) |
| **DX Coding** | None (camera defaults to ISO 100 or camera-specific default) |
| **Grain** | Fine (improved in 2017 and 2019 reformulations) |
| **Exposure Latitude** | Wide — usable from ISO 100 to 400 |
| **Development** | Standard C-41, no push/pull required |
| **Special Filters** | Not required (unlike Aerochrome which needed yellow/orange filters) |
| **Origin (2019)** | Made in China |
| **Origin (2013)** | Made in Europe (some early batches) |

## Color Transformation Map

Lomography's official description of the color shifts:

| Input Color | Output Color |
|-------------|-------------|
| **Green** | Purple / Magenta / Violet |
| **Blue** | Green / Teal |
| **Yellow** | Pink |
| **Red** | Red (preserved — skin tones remain relatively natural) |
| **Pink** | Yellow (bonus community discovery) |

The underlying mechanism is believed to be a **modified dye coupler arrangement** in the color layers — essentially shuffling which color layer produces which dye during development. One community analysis describes it as a Green/Blue channel swap effect in the emulsion.

## Formulation History

### Original 2013 Formula ("Old Purple")

- **Launch:** Announced January 2013, first rolls delivered July 2013
- **Initial run:** ~4,000 rolls in 120, ~3,500 rolls in 35mm
- **Characteristics:** Deep, dark, high-contrast purples and violets. Greens converted to deep purple-violet. Strong, dramatic color shifts. Higher contrast and darker overall rendering.
- **Reception:** Mixed — enthusiasts loved the dramatic shifts; skeptics found it too extreme
- **Price at launch:** ~$9 USD (35mm), ~$11 USD (120) — roughly double standard color negative film

### 2017 Reformulation

- Announced by Lomography as improvements to sensitivity, grain, and red hue rendering
- "Increased the film's sensitivity to red hues, improved exposure at the recommended setting of ISO 400, and significantly reduced grain"
- DPReview coverage confirmed: "better red hues, improved sensitivity and finer grain — but don't worry — will still shift blues to green, greens to purple and yellows to pink"

### New 2019 Formula ("New Purple" / "Purple XR")

- **Current production formula** — all boxes since 2019 marked "New 2019 formula"
- **Characteristics:** Lighter, more pronounced tones compared to old formula's darker contrast. More saturated emulsion with stronger **magenta** hues (shifted from violet toward pink/magenta). Reduced grain, greater sharpness, increased sensitivity to red hues.
- **Community consensus:** Weaker/more subdued color shift than original. Greens shift toward pink-magenta rather than deep violet-purple. "More refined but less dramatic."
- **ISO effect:** ISO 100 = pale/shiny purple greens; ISO 200 = balanced; ISO 400 = darker indigo, more intense red and purple

### 2021 "Pétillant" Formula

- A variant noted in some Lomography magazine articles
- Described as having "more contrasts and shadows"
- Appears to be a limited regional variant or rebranding

## ISO Behavior

Lomochrome Purple is a **variable ISO** film — results vary significantly by rated speed:

- **ISO 100:** Paler, muted color shifts. Greens → shiny purple. Lower contrast. Good for bright sunlight.
- **ISO 200:** Balanced between intensity and subtlety. Recommended sweet spot by many shooters.
- **ISO 400:** Most intense color shifts. Greens → dark indigo. Stronger reds and purples. More contrast. Best for overcast/low light.
- **Low light / flash:** Color shift "magic" diminishes significantly. Muddy, dull results reported.

## Film Base / Manufacturer Speculation

- 2019 formula marked "Made in China" on packaging
- Original 2013 some marked "Made in Europe"
- Community speculation points to finishing/confectioning in China (possibly China Lucky Film, though Lucky reportedly stopped color film production years ago)
- Exact manufacturer remains a Lomography trade secret
- Not simply a respooled consumer film with processing tricks — the color shifts occur at the emulsion level

## Comparison: LomoChrome Purple vs Kodak Aerochrome

| Aspect | Lomochrome Purple | Kodak Aerochrome (EIR) |
|--------|-------------------|------------------------|
| **Type** | C-41 color negative | E-6/AR-5 color reversal (slide) |
| **IR Sensitivity** | None — purely chemical color shift | True infrared sensitivity (layer sensitive to IR) |
| **Filters needed** | None | Yellow #12 or #15 typically required |
| **Green foliage result** | Purple/magenta | Magenta/red |
| **Availability** | In production (2025) | Discontinued 2011, $300-400/roll on secondary market |
| **Skin tones** | Natural-ish (rosy) | Unnatural (typically yellow) |
| **Grain** | Fine | Very fine (slide film) |
