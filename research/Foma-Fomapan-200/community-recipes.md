# Foma Fomapan 200 — Community Recipes

## Community Notes (from forums)
- No dedicated Fuji X Weekly recipe exists for Fomapan 200
- Photrio discussions note Fomapan 200 behaves differently from 100/400 — the hybrid emulsion responds uniquely
- The T-grain component provides finer grain than expected for 200-speed
- Some forum users report quality control issues with certain batches (emulsion defects)
- Reddit r/analog: less popular than Fomapan 100/400 but has a dedicated following
- Compared to Ilford FP4+ (125) by some users — similar sharpness, Fomapan has more grain character

## LR Preset Design Notes
- Hybrid T-grain/cubic grain: GrainAmount=35, Size=28, Roughness=55
  - T-grain component = smaller grain size than pure cubic
  - Cubic component = more character than pure T-grain
- Moderate contrast (+22) — slightly more than Fomapan 100
- Standard panchromatic GrayMixer: Red +20, Orange +15, Yellow +5, Blue -25
- Slightly extended blue sensitivity: slightly brighter greens (+5)
- Blacks -22 (slightly less aggressive than Fomapan 100)
- Moderate highlight reduction (-18)
- All STYLEGUIDE rules: Sharpness=10 with grain, Clarity/Texture/Dehaze=0, no Calibration, no WB
- The hybrid grain structure creates a "best of both worlds" character
