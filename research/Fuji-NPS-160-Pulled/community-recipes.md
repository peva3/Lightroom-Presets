# Fuji NPS 160 Pulled — Lightroom Preset Settings & Recipes

> Compiled from Fuji X Weekly (Thomas Schwab / Ritchie Roesch, May 2021), Reddit (r/analog, r/Lightroom), and photography forums. The pulled variant creates an ultra-soft, ethereal portrait aesthetic.

---

## Fuji X Weekly Recipe (May 2021)

**Source**: `https://fujixweekly.com/2021/05/30/fujifilm-x-trans-iv-x-pro3-x100v-x-t4-x-s10-x-e4-film-simulation-recipe-fujicolor-nps-160-pulled/`

Created by Thomas Schwab (effzwo.thommy on Instagram), with sample photos also by Ritchie Roesch.

```
Film Simulation: Classic Negative
Dynamic Range: DR400
Highlight: -2
Shadow: -2
Color: -4
Noise Reduction: -4
Sharpness: +1
Clarity: -2
Grain Effect: Weak, Small
Color Chrome Effect: Weak
Color Chrome Effect Blue: Strong
White Balance: Auto, -1 Red & -4 Blue
ISO: Auto, up to ISO 6400
Exposure Compensation: +2/3 to +1 1/3 (typically)
```

### Translation to Lightroom Concepts

| Fuji Setting | Lightroom Approximate |
|-------------|----------------------|
| Classic Negative base | Green-shadow bias, vintage muted palette |
| Highlight -2 | Strong highlight protection (-50) |
| Shadow -2 | Lifted shadows (+40) |
| Color -4 | Significant desaturation (-15) |
| WB Auto -1R -4B | Cooler auto white balance |
| Grain: Weak, Small | Light fine grain (Amount 18, Size 18) |
| Clarity -2 | Mild soft bloom |
| Color Chrome Effect Blue: Strong | Deepened blue tones |
| EC +2/3 to +1 1/3 | Significant overexposure (+0.80) |

## Community Consensus

### Defining Characteristics
1. **Pull processing aesthetic** — overexposed + underdeveloped = ultra-soft
2. **Very desaturated** — Color -4 on Classic Negative is extremely muted
3. **Green-cyan shadows** — Classic Negative's signature shadow cast
4. **Extended highlight range** — DR400 + Highlight -2 + pull = massive highlight latitude
5. **Fine grain** — Weak, Small grain on low ISO base

### Lightroom Community Values

From Reddit and community discussions:

```
Exposure: +0.65 to +1.00 (pull = overexposure)
Contrast: -20 to -30 (very low)
Highlights: -45 to -60 (strong protection)
Shadows: +30 to +45 (very lifted)
Blacks: +12 to +20
Saturation: -12 to -18
Vibrance: -8 to -14 (within 5 of Saturation)
Color Grading Shadows: Green-cyan (Hue 155-170°, Sat 8-12)
Grain: Amount 15-22, Size 15-22
```
